📝 How to use placeholders

Use {{field_name}} to insert contact-specific data into each message.

Built-in fields (always available):
{{name}} {{phone}} {{company}}
Custom fields (from CSV columns):

Any extra column in your imported CSV becomes a custom field. For example, if your CSV has columns like city, product, or designation, simply use them as:

{{city}} {{product}} {{designation}}
Detected custom fields (from selected group):
Example:
Hi {{name}}, your order for {{product}} is confirmed. We'll deliver to {{city}}. Thanks! — {{company}}
